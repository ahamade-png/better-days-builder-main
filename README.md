# Better Days Builder

## Local development

Requirements:

- Node.js
- npm

Install dependencies and start the app:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Notes

- Booking links are configured through `VITE_BOOKING_URL`.
- Book of Business authentication is handled through Supabase Auth.
